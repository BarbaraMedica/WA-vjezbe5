# WA-vjezbe5  
_## Pokretanje backend-a_  

**1. Instaliraj dependencies:**  
cd pizza-express  
npm install  

**2 Napravi env. datoteku:**  
*ona sadrži:  
  *MONGO_URI=<upišite svoj MongoDB connection string>   
  *MONGO_DB_NAME=<ime baze podataka>  
**3. Pokrenite server u terminalu tako da upišete:**  
  *node index.js    

_## Pokretanje frontend-a_  
1. Instaliraj dependencies:  
cd pizza-vue  
npm install  

**2.** Pokrenite upisujući **vite** u terminalu.
